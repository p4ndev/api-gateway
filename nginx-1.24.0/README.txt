# Installation - Windows
http://nginx.org/en/docs/windows.html

# Start / Stop / Quit
`start nginx`
`nginx -s reload`
`nginx -s stop`
`nginx -s quit`

# Using a different port
`nginx.conf` and change the `server > listen` port
